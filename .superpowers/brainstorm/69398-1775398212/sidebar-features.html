Which features belong in the plugin sidebar?

Select all that feel right for v1. You can click multiple options.

1

AI Chat (main view)

Already exists. The core experience — talk to AI to control your DAW. This is the default screen.

2

Connection Status

Live indicator showing plugin ↔ cloud ↔ DAW bridge status. See if your FL Studio bridge is connected, cloud relay is online, latency info.

3

Project Info

Current DAW project snapshot — BPM, track list, playing state. Quick read-only view of what the AI "sees" about your session.

4

Quick Mixer Controls

Visual sliders/knobs for track volume, pan, mute, solo — a mini mixer you can tweak without typing commands.

5

Chat History / Sessions

Browse past conversations. Resume previous sessions. See what commands were run before.

6

AI Presets / Quick Actions

Saved prompt templates — "Set up a lo-fi beat", "Add sidechain compression", "Analyze my mix". One-click commands.

7

Settings

Plugin preferences — theme, AI model selection, notification preferences, token/auth management, sign out.

8

Account & Subscription

Current plan (Free/Pro/Studio), usage stats, upgrade prompts. Manage billing without leaving the plugin.

9

Help & Onboarding

Getting started guide, what the AI can do, example commands, keyboard shortcuts, support link.
