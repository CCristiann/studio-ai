Sidebar layout for 1100×800 plugin

Space is limited inside a DAW. Which approach fits best?

A

Icon Rail (collapsed)

Narrow strip (~48-56px) with icon-only buttons. Always visible. Tooltips on hover. Think VS Code activity bar or Discord. Maximizes chat space.

B

Icon Rail + Expandable Panel

Same icon rail, but clicking an icon slides open a panel (like VS Code sidebar). Chat shrinks to accommodate. Click again to collapse back to icons-only.

C

Top Tab Bar

Horizontal tabs along the top — Chat | Status | Presets | Settings. Full width for each view. Simple but uses vertical space (~40px).

D

Bottom Navigation Bar

Mobile-style bottom bar with icons + labels. 5 items max. Leaves top clean for content. Common in embedded UIs.
